### Quickstart

```bash
$ git clone https://github.com/powerticket/ticket-the-pirate.git
$ cd ticket-the-pirate
$ ./gradlew clean build
$ java -jar build/libs/*.jar
```

or using docker

```bash
$ docker run -dp 8080:8080 powerticket/ticket-the-pirate
```
